Sobre el Projecte

Aquest projecte consisteix en el desplegament d'una aplicació web d'una agència de viatges utilitzant Railway.app i GitHub. L'objectiu principal és demostrar com es pot automatitzar el procés de desplegament d'aplicacions web en un entorn professional.

Característiques principals:

  • Desenvolupat amb Node.js
  • Integració contínua amb GitHub
  • Desplegament automàtic a Railway.app
  • Gestió eficient d'actualitzacions
